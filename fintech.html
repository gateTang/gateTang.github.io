After learning about the technicality in investments, specifically in topics like portfolio daily returns and standard deviation, I have been interested in creating investment strategies that can help less-experienced-investors find the correct indicators and assets to invest in. This will mainly be done so through data and quantitative analysis tools like:

  • Modern Portfolio Theory (Minimum Variance Efficient Frontier, portfolio daily returns and standard deviations).
  • Denoising Tools (Simple Moving Averages)
  • Equity Curves
  • Drawdown Charts

These tools are used to find the best combination of assets in portfolio. This may range from ETFs, bonds, and stocks in a portfolio. Currently, I am working on finding the best 3-asset combination.
